

O repositório está organizado na seguintes partes:

1 - O arquivo main.py contém a parte principal da tarefa;
2 - O arquivo EnviarCodigoInternet.py contém apenas o envio de conteúdo para o pastebin;
3 - O arquivo ContarZIP.py realiza a contagem de tamanhos dos arquivos zipados;
3 - A pasta "data" contém os arquivos:
    -JSON
    -Pickle
    -Gráfico relativo à tarefa
    -Relatório da tarefa
    -Somas_tamanhos é o arquivo JSON que contém os tamanhos dos ZIPs de todos os anos
